##### 3.Mac显示隐藏文件夹

> 最近查看Typora主题文件时候，发现文件被隐藏了，所以需要这个功能

```markdown
- Typora主题文件路径：
- /Users/gongwei/Library/Application Support/abnerworks.Typora/themes


- 同时按住“ Command + Shift + . ”三个按键，即可查看当下文件夹中的隐藏文件

- 若想恢复隐藏文件的话再次按“ Command + Shift + . ”，即可恢复文件的隐藏状态
```
